<h3>소개</h3>
GemmyLib는 오픈소스 스피갓 플러그인 개발 라이브러리입나다. 이는 다음과 같은 기능들을 갖고 있습니다.

- 여러 버전의 NMS의 쉬운 사용
- 쉬운 모듈 추가
- 코틀린에 최적화된 코드
- 쉬운 게임 내 플레이어 알림 시스템
- 기본 모듈
    1. 블록 시스템
    2. 클라이언트에서만 존재하는 가상 엔티티 관리
    3. 가상 표지판을 사용한 입출력 시스템
    4. 쉬운 플레이어 수정 시스템
- 명령어를 쉽게 등록할 수 있는 커맨드 관리 시스템
- 간단한 인벤토리 관리 시스템
  <br /><br />

GemmyLib는 JDK 16 과 Kotlin 1.5.20 버전을 사용하며 Kolleague에 의존합니다.
